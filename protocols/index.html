Protocol Stack Overview

Application Layer: COSEM (Object Model, Methods, Attributes)
DLMS Layer: ACSE, xDLMS (GET, SET, ACTION, etc.)
Data Link Layer: HDLC, Wrapper, M-Bus
Physical Layer: Serial, Optical, TCP/IP, PLC, GSM/GPRS
