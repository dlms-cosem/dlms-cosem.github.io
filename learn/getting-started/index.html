Getting Started with DLMS/COSEM Development

Ready to start building with DLMS/COSEM? This guide will help you set up your development environment and connect to your first meter (or simulator).

‚úÖ

You don't need physical hardware to start! We'll use the free Gurux DLMS Simulator.

Prerequisites

Required Knowledge

  • Basic programming (Python, JavaScript, or C#)
  • Understanding of networking concepts
  • Familiarity with command line

System Requirements

  • OS: Windows, Linux, or macOS
  • RAM: 4GB minimum, 8GB recommended
  • Disk: 1GB free space
  • Network: For remote meter access

Development Environment Setup

Your First Connection

Python Example: Connect to Simulator

# first_connection.py
from dlms_cosem import cosem
from dlms_cosem.io import TcpTransport

# Create transport (TCP/IP to simulator)
transport = TcpTransport(
    host='localhost',
    port=4059
)

# Create DLMS client
client = cosem.DlmsClient(
    transport=transport,
    authentication='none'  # No auth for testing
)

try:
    # Connect to simulator
    print("Connecting to simulator...")
    client.connect()
    print("‚úì Connected successfully!")

    # Read meter serial number (OBIS: 0.0.96.1.0.255)
    serial = client.get(
        class_id=1,  # Data object
        obis='0.0.96.1.0.255',
        attribute_id=2  # Value attribute
    )
    print(f"Meter Serial: {serial}")

    # Disconnect
    client.disconnect()
    print("‚úì Disconnected")

except Exception as e:
    print(f"‚úó Error: {e}")
finally:
    transport.close()

Run it:

python first_connection.py

C# Example: Connect to Simulator

using Gurux.DLMS;
using Gurux.Net;
using System;

class Program
{
    static void Main()
    {
        // Create TCP/IP connection
        var media = new GXNet()
        {
            HostName = "localhost",
            Port = 4059,
            Protocol = NetworkType.Tcp
        };

        // Create DLMS client
        var client = new GXDLMSClient(
            useLogicalNameReferencing: true,
            clientAddress: 16,
            serverAddress: 1,
            authentication: Authentication.None,
            password: null,
            interfaceType: InterfaceType.WRAPPER
        );

        try
        {
            // Open connection
            Console.WriteLine("Connecting...");
            media.Open();

            // Send association request
            var data = client.SNRMRequest();
            media.Send(data, null);
            media.Receive(data);
            client.ParseUAResponse(data);

            // Request association
            data = client.AARQRequest();
            media.Send(data, null);
            media.Receive(data);
            client.ParseAAREResponse(data);

            Console.WriteLine("‚úì Connected!");

            // Read serial number
            var obj = new GXDLMSData("0.0.96.1.0.255");
            data = client.Read(obj, 2)[0];
            media.Send(data, null);
            media.Receive(data);
            client.UpdateValue(obj, 2, data);

            Console.WriteLine($"Serial: {obj.Value}");

            // Disconnect
            media.Close();
            Console.WriteLine("‚úì Disconnected");
        }
        catch (Exception ex)
        {
            Console.WriteLine($"‚úó Error: {ex.Message}");
        }
    }
}

Reading Common Values

Read Instantaneous Power

# Read active power (OBIS: 1.0.1.7.0.255)
power = client.get(
    class_id=3,  # Register
    obis='1.0.1.7.0.255',
    attribute_id=2  # Value
)

print(f"Active Power: {power} W")

Read Energy Consumption

# Read total energy (OBIS: 1.0.1.8.0.255)
energy_data = client.get(
    class_id=3,  # Register
    obis='1.0.1.8.0.255',
    attribute_id=2
)

# Get scaler and unit
scaler_unit = client.get(
    class_id=3,
    obis='1.0.1.8.0.255',
    attribute_id=3
)

# Calculate actual value
value = energy_data * (10 ** scaler_unit['scaler'])
unit = scaler_unit['unit']  # 30 = Wh

print(f"Total Energy: {value} Wh ({value/1000} kWh)")

Read Voltage (3-Phase)

# Read all three phase voltages
voltages = {}
for phase, obis in [
    ('L1', '1.0.32.7.0.255'),
    ('L2', '1.0.52.7.0.255'),
    ('L3', '1.0.72.7.0.255')
]:
    voltage = client.get(
        class_id=3,
        obis=obis,
        attribute_id=2
    )
    voltages[phase] = voltage
    print(f"Voltage {phase}: {voltage} V")

Common Issues and Solutions

Issue: Connection Refused

Symptoms:

Error: Connection refused

Solutions:

  • Check if simulator is running
  • Verify port 4059 is correct
  • Check firewall settings
  • Try telnet localhost 4059 to test

Issue: Authentication Failed

Symptoms:

Error: Access denied / Authentication failed

Solutions:

  • Check authentication level (none/LLS/HLS)
  • Verify password/key if using authentication
  • Confirm client address matches server configuration

Issue: Timeout

Symptoms:

Error: Timeout waiting for response

Solutions:

  • Increase timeout value
  • Check network connectivity
  • Verify server is responding
  • Check if correct communication profile is used

Issue: Invalid OBIS Code

Symptoms:

Error: Object not found / Access denied

Solutions:

  • Verify OBIS code format (A.B.C.D.E.F)
  • Check if object exists in meter
  • Confirm access rights for your client
  • Read association object to list available objects
