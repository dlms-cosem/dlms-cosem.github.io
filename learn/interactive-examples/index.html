Common OBIS Code Examples

Electricity Metering

Try these common electricity meter OBIS codes:

Energy Registers:

  • 1.0.1.8.0.255 - Total active energy import (kWh)
  • 1.0.2.8.0.255 - Total active energy export (kWh)
  • 1.0.3.8.0.255 - Total reactive energy import (kVARh)

Instantaneous Values:

  • 1.0.1.7.0.255 - Active power import (kW)
  • 1.0.32.7.0.255 - Voltage phase L1 (V)
  • 1.0.31.7.0.255 - Current phase L1 (A)
  • 1.0.14.7.0.255 - Frequency (Hz)
  • 1.0.13.7.0.255 - Power factor

Three-Phase Values:

  • 1.0.52.7.0.255 - Voltage phase L2
  • 1.0.72.7.0.255 - Voltage phase L3
  • 1.0.51.7.0.255 - Current phase L2
  • 1.0.71.7.0.255 - Current phase L3

Abstract Objects

System and configuration objects:

  • 0.0.1.0.0.255 - Clock
  • 0.0.96.1.0.255 - Meter serial number
  • 0.0.96.1.1.255 - Manufacturer ID
  • 0.0.42.0.0.255 - Logical device name

Gas Metering

Gas meter OBIS codes:

  • 7.0.3.0.0.255 - Volume (total)
  • 7.0.3.7.0.255 - Instantaneous volume flow
  • 7.0.11.0.0.255 - Temperature
  • 7.0.42.0.0.255 - Pressure

Water Metering

Water meter OBIS codes:

  • 8.0.1.0.0.255 - Water volume (total)
  • 8.0.1.7.0.255 - Instantaneous flow rate
  • 8.0.3.0.0.255 - Forward volume
  • 8.0.4.0.0.255 - Reverse volume (leakage)

JavaScript/TypeScript Example

Using the @dlms-cosem/parser library:

import { parseObis, validateObisCode } from '@dlms-cosem/parser';

// Parse an OBIS code
const info = parseObis('1.0.1.8.0.255');

console.log(info.metadata.description);
// Output: "Total Active Energy Import (kWh)"

console.log(`Medium: ${info.metadata.mediumName}`);
// Output: "Medium: Electricity"

console.log(`Quantity: ${info.metadata.quantityName}`);
// Output: "Quantity: Active Power/Energy"

console.log(`Unit: ${info.metadata.unit}`);
// Output: "Unit: W/Wh"

// Validate an OBIS code

Building a Simple Meter Dashboard

Here's how to build a basic real-time meter dashboard:

Step 1: Define the OBIS Codes to Read

const dashboardObjects = [
  '1.0.1.8.0.255',  // Total energy (kWh)
  '1.0.1.7.0.255',  // Current power (kW)
  '1.0.32.7.0.255', // Voltage L1 (V)
  '1.0.52.7.0.255', // Voltage L2 (V)
  '1.0.72.7.0.255', // Voltage L3 (V)
  '1.0.31.7.0.255', // Current L1 (A)
  '1.0.51.7.0.255', // Current L2 (A)
  '1.0.71.7.0.255', // Current L3 (A)
  '1.0.14.7.0.255', // Frequency (Hz)
  '1.0.13.7.0.255', // Power factor
];

Step 2: Read Values from Meter

import { parseObis } from '@dlms-cosem/parser';

async function readMeterData(client) {
  const values = {};

  for (const obis of dashboardObjects) {
    try {
      // Read value from meter
      const value = await client.get(obis);

      // Parse OBIS code for metadata
      const info = parseObis(obis);

      values[obis] = {
        value: value,
        description: info.metadata.description,
        unit: info.metadata.unit,
      };
    } catch (error) {
      console.error(`Error reading ${obis}:`, error);
    }
  }

  return values;
}

Step 3: Display the Data

function displayDashboard(values) {
  console.log('\n=== Meter Dashboard ===\n');

  for (const [obis, data] of Object.entries(values)) {
    const valueStr = `${data.value} ${data.unit || ''}`;
    console.log(`${data.description}: ${valueStr}`);
  }
}

// Update every 5 seconds
setInterval(async () => {
  const data = await readMeterData(client);
  displayDashboard(data);
}, 5000);

Load Profile Analysis

Reading and analyzing historical load profile data:

// OBIS code for load profile (1.0.99.1.0.255)
const LOAD_PROFILE_OBIS = '1.0.99.1.0.255';

async function getLoadProfile(client, fromDate, toDate) {
  // Read load profile entries
  const profile = await client.getLoadProfile(
    LOAD_PROFILE_OBIS,
    fromDate,
    toDate
  );

  return profile.map(entry => ({
    timestamp: entry.timestamp,
    activePower: entry.values['1.0.1.7.0.255'],
    voltage: entry.values['1.0.32.7.0.255'],
    current: entry.values['1.0.31.7.0.255'],
  }));
}

// Get last 24 hours
const endDate = new Date();
const startDate = new Date(endDate - 24 * 60 * 60 * 1000);

const profile = await getLoadProfile(client, startDate, endDate);

// Calculate statistics
const avgPower = profile.reduce((sum, p) => sum + p.activePower, 0) / profile.length;
const maxPower = Math.max(...profile.map(p => p.activePower));
const minPower = Math.min(...profile.map(p => p.activePower));

console.log(`Average Power: ${avgPower.toFixed(2)} kW`);
console.log(`Peak Power: ${maxPower.toFixed(2)} kW`);
console.log(`Minimum Power: ${minPower.toFixed(2)} kW`);
